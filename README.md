# Project3_EddieVargas
This repository contains Project #3 for CIVE 202, using NHTS and NGSIM data to visualize transportation trends and driving behavior. It includes bar, histogram, boxplot, and time-series plots, along with an IDM simulation to compare real and simulated vehicle behavior.

User Guide: Open the Jupyter Notebook (.ipynb) and run all cells sequentially. Ensure required libraries (pandas, matplotlib, seaborn) are installed. The code will load the datasets, generate visualizations, and run the IDM simulation automatically.
